# Formula1HistoricalDataPipeline-Dashboard_-WIP-
Proyecto de análisis de datos de Fórmula 1 desarrollado con Python y Power BI. El proyecto automatiza la recopilación de resultados históricos desde Formula1.com, realiza procesos de limpieza y transformación de datos, genera un dataset estructurado en Excel y lo utiliza para crear un dashboard interactivo en Power BI.
